# Nexo Visual - Site de Portfólio

Site estático do estúdio Nexo Visual, desenvolvido em HTML5, CSS3 e JavaScript vanilla.

## Arquivos

- `index.html`: Página principal com todas as seções
- `styles.css`: Estilos CSS com variáveis, responsividade e otimizações mobile
- `script.js`: JavaScript para interações (navbar, menu mobile, filtros)
- `logo.png`: Logo em alta resolução (2400x1600px)
- `404.html`: Página de erro personalizada
- `README.md`: Este arquivo

## Como Visualizar

Abra o arquivo `index.html` em qualquer navegador web moderno.

## Funcionalidades

- ✓ Navegação fixa com efeito de scroll
- ✓ Menu hamburger funcional em mobile (clique para abrir/fechar)
- ✓ Layout responsivo otimizado para smartphones (320px, 480px, 768px+)
- ✓ Seções: Hero, Sobre, Serviços, Portfólio com filtros, Processo, Depoimentos, Contato
- ✓ Formulário de contato (frontend)
- ✓ Acessibilidade melhorada em dispositivos móveis
- ✓ Logo em alta resolução (PNG 2400x1600px)

## Melhorias Mobile

- Menu hamburger com animação (transforma em X ao abrir)
- Menu se fecha automaticamente ao clicar em um link ou fora dele
- Espaçamento adequado para toques confortáveis
- Tipografia ajustada para leitura em telas pequenas
- Grids adaptáveis para uma coluna em mobile
- Padding e margins otimizados para cada breakpoint
- Logo responsiva nos breakpoints

## Tecnologias

- HTML5
- CSS3 (variáveis CSS, media queries)
- JavaScript ES6
- Google Fonts: Syne, DM Sans, DM Mono
